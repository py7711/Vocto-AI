# Dashboard Transcriptions Specification

## Overview

- Target file: `src/components/workspace/Workspace.tsx`
- Screenshot: `docs/design-references/uniscribe.co/dashboard-desktop.png`
- Interaction model: default browsing mode plus click-driven bulk mode.

## Live Content

- Heading: `All Transcriptions`
- Search placeholder: `Search file name`
- Default buttons: `Upload Files`, `Paste Link`, `Bulk Actions`
- Bulk buttons: `Move`, `Export`, `Delete`, `Cancel`
- Table columns: `Name`, `Duration`, `Created`, `Type`, `Folder`
- Rows per page: `10`
- Promotion card upgrade prompt title: `Upgrade Your Plan`
- Promotion card full pricing overlay title: `Plans & Pricing`

## Behavior

- Default mode shows normal row links and row action buttons.
- Sidebar `Upgrade Plan` and promotion-card `Upgrade Now` open the target compact dashboard upsell dialog in place on `/dashboard`; they do not navigate to `/pricing`.
- Account menu theme buttons are functional global theme controls. `Light` applies the root `light` class, `Dark` applies the root `dark` class, and `System` resolves from `prefers-color-scheme` while retaining the target selected-button geometry.
- Compact upsell dialog copy is `Upgrade Your Plan`, `Unlock more transcription time and premium features to boost your workflow`, `SUBSCRIPTION`, `Upgrade to Basic Plan Now`, `1200 minutes of transcription per month`, `Premium transcription model (highest accuracy)`, `Each file can be up to 10 hours long / 5 GB. Upload 50 files at a time.`, `No daily file limit for transcription`, `Yearly Plan`, `$6/mo`, `Billed yearly ($72/year)`, `40% OFF`, `Monthly Plan`, `$10/mo`, `Cancel anytime`, `Upgrade Now`, `See All Plans`, `Cancel anytime`, and `Instant access`.
- Compact upsell `See All Plans` keeps the dashboard URL and swaps to the full-screen pricing overlay with `Annual` selected. Promotion-card `See All Plans` opens the full-screen pricing overlay directly with `Monthly` selected, matching the previously observed promotion-card pricing surface.
- Full-screen pricing overlay contains `One-Time`, `Monthly`, and `Annual` tabs, pricing cards, FAQ, and a top-right `Close` button.
- Pricing overlay billing tabs match the target `/pricing` tablist geometry: `512px` max width, `40px` height, `4px` padding, `6px` radius, slate-100 background, no outer border; each tab is `32px` tall with a `4px` radius.
- Target/local overlay vertical rhythm rechecked and verified on 2026-07-02: the tablist sits around `y=165`, the monthly card row starts around `y=237`, and `Subscribe now` CTAs sit at `y=378` at 1280x720. Local spacing was tightened to `mt-6` before the tablist, `mt-8` before the cards, and the checkout action wrapper uses zero top margin to match this target rhythm.
- Target rechecked on 2026-07-02: Monthly/Annual overlay plan CTAs (`Subscribe now`) are `button` elements with no `href`; the Standard CTA is violet, while Basic/Pro are white outline-style buttons. They remain in the overlay and trigger the subscription checkout flow rather than navigating to `/pricing`.
- Pricing overlay annual Basic card uses `$6 / month`, `$10` struck through, and `($72 / year, billed yearly)`.
- Pricing overlay cards use the same target card header as `/pricing`: plan name and tagline only, with no extra right-aligned badge/check icon.
- Pricing overlay feature rows include target-style `New` badges and `More information` icon buttons for AI translation / AI Insights, and `API access` links to `/docs`.
- Pricing overlay FAQ uses the same target plain `divide-y` list as `/pricing`: no rounded bordered FAQ card, 718px desktop width, and all questions collapsed by default.
- `Upload Files` is rendered as a button-style control that navigates to `/upload`; `Paste Link` is rendered as a button that keeps the dashboard URL and opens `Media Link Transcription` in place.
- Promotion card timer displays a two-digit `MM:SS:CS` countdown. Target re-measured on 2026-07-01 with samples such as `01:34:95`, `00:52:16`, `00:49:69`, and `00:45:47`; the final segment behaves like centiseconds and can exceed `59`.
- Dashboard `Paste Link` opens the same default `Media Link Transcription` modal pattern as Upload. At 1280x720 it is centered at `x=320`, `y=172`, `640x376`, with a 1px slate-200 border, 8px radius, no visible shadow, and a top-right icon-only `Close` button. It must not use the legacy 425px dialog or a bottom `Close` button.
- Default dashboard link modal content: title `Media Link Transcription`; helper copy `Paste a media link to transcribe video or audio content.`; supported-platform card `590x122` with slate-100/25 background and slate-200/60 border; chips are 30px high with platform icons for `YouTube`, `TikTok`, `Instagram`, `Facebook`, `X`, and `Many other links`.
- Default dashboard link modal input row: input accessible name `Link input`, placeholder `Paste a media link`, 40px high, flexes to about `481px`; disabled `Search` is text-only, about `97px` wide, white background, violet border/text, no search icon.
- Dashboard link modal footer is inside the dialog body as a `590px` row starting at y `477`; it shows `Available minutes:`, the remaining-minute value, and `Buy More Minutes` as one centered inline group at y `502`, with a 1px top divider, 8px top padding, and 16px inner top margin.
- Row action menu items are `Share`, `Export`, `Rename`, `Move`, separator, `Delete Transcription`.
- `Share` opens a centered `Share Transcription` dialog. Disabled state copy is `Sharing is currently disabled. Enable it to generate a public link.` with a single full-width `Enable sharing` button and an icon-only top-right `Close`; there is no bottom `Close` button in the disabled state. Enabled state displays the public link and allows copying or disabling sharing. `Disable sharing` opens an in-app confirmation; the final DELETE request is only executed from the confirmation dialog's red `Disable` action.
- `Rename` opens a centered `Rename File` dialog with one text input, placeholder `Enter new filename`, and `Cancel` / `Rename` buttons.
- `Export` opens a centered `Export Options` dialog with six formats: `txt`, `docx`, `pdf`, `csv`, `srt`, `vtt`; speaker and timestamp toggles; and a centered `Export` button.
- `Move` opens a centered `Move to Folder` dialog with current-folder copy and selectable folder rows.
- `Delete Transcription` opens a centered confirmation dialog. The final destructive delete is only executed from the dialog's red `Delete` action.
- Bulk mode is entered by clicking `Bulk Actions`.
- Bulk mode shows selected count and row/header checkboxes.
- Bulk `Move` opens a centered `Move to Folder` dialog with `Moving N selected files`, folder choices, and a disabled `Move` button until a destination is selected.
- Bulk `Export` opens the same centered `Export Options` dialog as row export, with helper text `Export N selected files`.
- Bulk `Delete` opens a centered `Delete Selected Files` alert dialog. The final destructive batch delete is only executed from the dialog's red `Delete Files` action.
- Bulk more menu exposes `Delete Originals`; it opens a centered confirmation dialog and only deletes original media from the red `Delete` action. Transcripts and generated content are preserved.
- Bulk mode exits through `Cancel`, and after successful batch move/delete/export.
- Row folders display as text in default mode.
